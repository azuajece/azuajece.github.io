# Carlos E. Azuaje - Portafolio Profesional

Portafolio profesional SPA (Single Page Application) moderno desarrollado con **React 19 + TypeScript + Vite**

## ✨ Características

- ⚡ **Vite** - Empaquetador ultra rápido
- ⚙️ **React 19** - Últimas características de React
- 📘 **TypeScript** - Tipado estricto y seguridad
- 🎨 **Diseño Responsivo** - Mobile-first
- 🚀 **GitHub Pages** - Fácil despliegue
- ♿ **Accesibilidad** - WCAG 2.1 compatible
- 🔄 **Performance** - Optimizado y rápido

## 📋 Secciones

1. **Sobre Mi** - Información profesional y aptitudes
2. **Curriculum** - Historial completo de experiencias laborales
3. **Contacto** - Formulario de contacto directo

## 🚀 Instalación y Uso

```bash
# Instalar dependencias
npm install

# Modo desarrollo
npm run dev

# Build para producción
npm run build

# Preview del build
npm run preview

# Deploy a GitHub Pages
npm run deploy
```

## ⚙️ Configuración

Antes de desplegar, actualiza `package.json`:

```json
"homepage": "https://tu-usuario.github.io/portafolio"
```

## 🏗️ Estructura del Proyecto

```
src/
├── components/
│   ├── Header.tsx        # Encabezado con perfil
│   ├── Navigation.tsx    # Navegación principal
│   ├── Header.css
│   └── Navigation.css
├── pages/
│   ├── SobreMi.tsx       # Sección Sobre Mi
│   ├── Curriculum.tsx    # Experiencias laborales
│   ├── Contacto.tsx      # Formulario contacto
├── styles/
│   ├── global.css        # Estilos globales
│   └── Pages.css         # Estilos de páginas
├── App.tsx               # Componente principal
├── App.css
└── main.tsx              # Entry point
```
